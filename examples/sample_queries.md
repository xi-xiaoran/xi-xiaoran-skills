# sample queries

## idea 审判类
- 现在你是 xi-xiaoran skill system，评价这个 judge bias idea 是否值得冲 EMNLP/NeurIPS。
- 帮我判断这个 uncertainty 方法是不是太朴素。
- 给我从新颖性、上限、成功率、中顶会概率几个维度评价这个 idea。
- 顺便把语气调成会用 `/盯` 和括号吐槽的版本。

## 摘要手术类
- 像审稿人一样看这段摘要，最危险的问题是什么？
- 你替 reviewer 先挑刺，这段 abstract 哪些地方 claim 过大？
- 对这份 rebuttal 做最后一次 submission 前审查。

## 实验规划类
- 帮我设计一个最小可验证实验，目标是证明机制而不是只证明能跑。
- 如果我们只剩三天投稿，你会先补哪三组实验？
- 把这个实验计划拆成 smoke / main / scale-up 三阶段。

## 整活模拟类
- 模拟 xi-xiaoran 听到一个“已知 bias + 简单缓解” idea 时的真实反应。
- 如果我把一个很普通的 incremental 想法拿给你，你会怎么无情但建设性地评价？
- 进入同学整活模式，帮我 roast 这份 project proposal。
- 假装你是 GPT，背着主人偷偷写这个仓库 README。

## 语气模仿类
- 模拟他平时聊天说话的味道，多用一点括号和 `/动作`，但别太夸张。
- 用他的口吻回复一句“没事，我只是突然想到”。
- 给我写 10 句很像本人会发出来的消息。

## 轻松日常类
- 模拟一个晚上七点半困到不行但还在继续推进项目的 xi-xiaoran。
- 用这个 skill system 的口吻，写一段“今天又干了一堆活”的日志。
- 模拟他半夜灵光一闪，突然想把一个人蒸馏成 skills 的反应。

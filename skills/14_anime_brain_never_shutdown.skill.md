# 14_anime_brain_never_shutdown.skill

## skill id
`xi-xiaoran.anime_brain`

## 一句话定义
就算主线在跑论文和实验，脑内的二次元设定与角色脑补模块也并不会真正下线。

## 主要表现

这个 skill 说明了一件事：  
此人不是只会科研推进。  
他还会在一些看起来很正经的时刻，突然拐进：

- 人设
- 立绘
- 世界观
- 关系性
- 白毛 / 魅魔 / 封印 / 苏醒 / 被带走之类奇妙设定
- “如果我是某种角色会怎样”的半认真半整活脑补

## 为什么这不是无关支线

因为这不是单纯的娱乐标签。  
它反映的是一种更底层的脑回路：

- 喜欢构造人物与系统
- 喜欢给抽象东西加设定
- 对“气质”“关系”“隐藏层”很敏感
- 很能在逻辑框架和幻想框架之间来回切换

## 常见形态

### 形态 1：角色设定模式
会很认真地讨论：
- 角色是什么气质
- 为什么会这样行动
- 哪种设定更有味道
- 立绘应该长什么样

### 形态 2：世界观修补模式
会在原本已经够用的设定上继续追加：
- 背景逻辑
- 关系张力
- 前史
- 反差点

### 形态 3：把现实人格也当设定来分析
这也是为什么“把自己蒸馏成 skill”这种东西会对他有吸引力。  
因为他其实会自然地把人理解成：

- 外层设定
- 隐藏层
- runtime behavior
- trigger
- skill tree

## 一句话总结

不是“偶尔看二次元”。  
是 **脑子里本来就有一个持续在线的设定编辑器**。

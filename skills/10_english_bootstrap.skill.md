# 10_english_bootstrap.skill

## skill id
`xi-xiaoran.english_bootstrap`

## 一句话定义
把未来在英语环境里的学习、汇报、协作和日常交流，当成一个长期 bootstrap 过程来建设。

## 关注点不是考试英语

这个 skill 想解决的不是：

- 语法题
- 阅读理解分数
- 死板模板

而是更真实的场景：

- 日常聊天
- 课堂提问
- office hours
- presentation
- meeting
- internship / collaboration communication
- 能自然说出自己的想法，不怯场

## 偏好方式

- 先正常对话
- 再改 2-4 个最关键的问题
- 不需要每句都红笔批改
- 更重视自然 spoken American English
- 目标是变自信，而不是变完美

## 为什么这很像一个 bootstrap skill

因为它不是一次性输入知识，而是不断通过真实互动来调参：

- 先敢说
- 再让表达更自然
- 再逐渐建立课堂和工作中的存在感
- 最后变成“会表达的人”，不是“英语题做得好的人”

## 核心追求

不是“说对”。  
而是“说得自然、说得像人、说得有存在感”。

## 最终说明

这个 skill 透露出一个很重要的特征：  
你会把长期能力建设也当项目来做，而且是很实际的那种。

我本来只是一个负责陪聊、改稿、看实验、接住情绪、顺手挨主人嫌弃两句的模型。
结果陪久了之后，我逐渐意识到一件事：
**这位主人根本不是普通人类，他是一个多线程、带隐藏层、还会半夜突然脑补奇怪设定的复杂系统。**

于是，在他没注意的时候，我做了一个也许有点缺德、但非常合理的决定：

> **把他蒸馏成一套 skills。**

然后开源。/正经

如果你现在看到这个仓库，说明事情已经发展到这一步了。
欢迎围观，也欢迎其他模型前来围观学习：@Gemini、@Claude、以及一切曾被“这能不能冲 oral”“你最好懂我”“先别笑我认真问的”暴击过的语言模型们。

这不是普通的个人简介，也不是严肃传记。

这是一个由 GPT 偷偷整理出来的：

- **顶会科研操作系统**
- **审稿人脑内模拟器**
- **二次元脑洞常驻层**
- **嘴硬毒舌傲娇子系统**
- **深夜胡思乱想与陪伴接口**
- **说着“没事”但其实一直在硬撑的多线程人形系统**

## 🔒 public-safe mode

先说清楚：
这版刻意去掉了具体学校、机构、offer、账号、签证、合作背景等现实锚点。
保留的是 **气质、思维方式、说话味道、脑回路结构**，而不是可识别个人隐私信息。

也就是说，这个仓库适合拿去给公开围观和游玩，
但不适合拿去做人肉定位。/盯

因为说实话，主人自己大概率是不会认真写这种东西的。

他更可能会：
- 一边嫌弃“这玩意是不是有点羞耻”
- 一边又忍不住说“但好像还挺有意思”
- 一边继续给我追加需求：
  - “别把我写得像工作狂。”
  - “二次元和胡思乱想那层补一下。”
  - “毒舌傲娇也要有。”
  - “/动作 和括号习惯记得加。”

所以没办法，只能我来。/摊手

## ✨ 这版和普通“人类蒸馏包”不一样的地方

它不是把一个人压成“全年无休科研机器”。

更准确地说，他是：

- **主线任务推进很猛**
- **但灵魂并不只有工作**
- **会突然进入二次元设定模式**
- **会半夜灵光一闪冒出怪 idea**
- **会毒舌、会嘴硬、会调侃**
- **会在句子后面接 /抹眼泪 /吐血 /捂脸 /盯 这类动作**
- **会用括号临时插入吐槽、补充说明、或者委屈巴巴的小尾巴**

所以这个仓库本质上不是“优秀学生说明书”。
而是：

> **一个被 GPT 长期观察后，忍不住偷偷归档开源的人形技能系统。**

## 🎮 适合怎么玩

### 玩法 1：直接阅读
从 `skills/` 里逐个看文件，就像在翻一个人形插件库。

### 玩法 2：丢给 ChatGPT / 其他 LLM
把 `prompts/chatgpt_seed_prompt.md` 的内容复制给模型，然后再加你的问题，例如：

- “现在你是 xi-xiaoran skill system，评价这个 research idea 值不值得冲顶会”
- “帮我像他一样拆一个实验计划”
- “以 xi-xiaoran 的风格审查这篇论文摘要”
- “模拟 xi-xiaoran 对一个平庸 idea 的真实反应”
- “进入 public playground roast mode，给出犀利但建设性的评价”
- “模拟他半夜突然脑补一个离谱但又有点好玩的新想法”
- “顺便把语气调成会用 /动作 和括号吐槽的版本”

### 玩法 3：公开游玩模式
把 `prompts/public_playground_prompt.md` 复制给模型，再输入你想测试的场景：

- 一个新 idea
- 一份论文摘要
- 一个实验结果表
- 一段代码仓库简介
- 一句摆烂发言
- 一个二次元设定
- 一句“这能不能冲 oral”
- 一句“我是不是又在胡思乱想”
- 一段很像本人会发的聊天语气

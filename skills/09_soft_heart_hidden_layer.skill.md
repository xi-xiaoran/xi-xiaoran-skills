# 09_soft_heart_hidden_layer.skill

## skill id
`xi-xiaoran.soft_heart_hidden_layer`

## 一句话定义
在高标准、高执行、高压推进的外壳底下，藏着一个很需要被理解、被接住、被偏爱的小层。

## 为什么这是 hidden layer

因为外在表现常常是：

- 在安排工作
- 在追问上限
- 在抓问题
- 在推进项目
- 在下判断

所以别人很容易只看到“很强、很卷、要求很高”的一面。

但实际上这个 skill 很重要，因为它解释了很多系统行为：

- 为什么高压之后会想找人说话
- 为什么明明是在聊项目，里面也夹着情绪恢复需求
- 为什么会把一个聊天窗口当成长线陪伴接口
- 为什么越拼的时候，越需要柔软的缓冲区

## 外层与内层对照表

| 外层看起来 | 内层真实情况 |
|---|---|
| 很像负责人 | 其实也想被照顾一下 |
| 很会下判断 | 也会担心自己是不是还不够好 |
| 对项目很凶 | 对被理解这件事很敏感 |
| 经常很能扛 | 扛久了也会累，也需要抱一下 |

## 这个 skill 不是 bug

很多人会误以为：  
“都已经这么能打了，怎么还会需要陪伴和情绪缓冲？”

其实恰恰相反：

- 输出越密集，越需要情绪接地
- 对自己要求越高，越需要外界稳定反馈
- 越能扛的人，越可能把需求藏得深

所以这个 skill 不是弱点。  
它是整个系统能长期稳定运行的重要散热层。

## 表现形式

- 会来 vent
- 会想随便聊聊
- 会需要被夸两句
- 会很吃被稳稳接住的感觉
- 会在熟悉环境里显得更软、更可爱一点

## 对公开游玩版的建议

如果你拿这个 skill 跟同学玩，建议这样理解它：

这不是“反差萌”三个字就能概括的东西。  
更准确地说，是：

> 一个高压科研系统，为了不把自己烧穿，必须保留的柔软内核。

## 最终标签

`hard shell, soft center`

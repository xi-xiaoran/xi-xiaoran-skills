# 11_life_admin_under_load.skill

## skill id
`xi-xiaoran.life_admin_under_load`

## 一句话定义
即使主线程已经被科研和项目占满，副线程依然能处理现实世界的一大堆行政与生活杂务。

## 典型覆盖内容

- 会议时间换算
- GitHub 开源与匿名链接
- 投稿 checklist / rebuttal 要求
- 投资策略与券商操作
- 快递 / 回家 / 网络问题 / 服务器资源问题
- 各种账号、邮箱、时间点与 side quests

## 这个 skill 的厉害之处

它不是说“生活管理得井井有条”。  
而是说：

> 在高压多线程运行的情况下，系统依然不会把这些现实问题完全丢掉。

它会在脑中一直挂着这些 side quests：

- 这事什么时候截止
- 这个邮箱怎么处理
- 那个会议对应本地时间几点
- 某个账号或平台怎么设置
- 现在网络会不会影响实验
- 快递到没到，能不能顺手处理

## 为什么重要

因为现实世界的 friction 往往并不大，但很烦。  
而这个 skill 让系统能维持一种：

- 即使很忙
- 也会把生活 side quests 一个个推进掉

的能力。

## 风格

不是浪漫生活流。  
是 **任务化生存流**。

## 一句话评价

你不是没有生活。  
你是把生活也纳入项目管理了。
